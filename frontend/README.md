# Frontend

- OS：Windows 10
- Node.js：v18.14.2
- npm：9.5.0
- vite：5.0.0

## App1

- create project

  ```
  npm config set strict-ssl false

  npm create vite@latest
  ```

- install

  ```
  npm install
  ```

- run dev

  ```
  npm run dev
  ```

- build

  ```
  npm run build:test
  npm run build:prod
  ```
